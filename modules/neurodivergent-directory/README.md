## The Problem

BIPOC adults are underdiagnosed, misdiagnosed, and face a critical lack of culturally affirming care. Existing directories:
- Don't filter by BIPOC-affirming providers
- Don't focus on late-diagnosis experience
- Lack real verification from BIPOC ND users

---

## The Solution

A searchable directory with real verification:
- **Providers:** Therapists, assessors, coaches (verified BIPOC-affirming)
- **Employers:** ND-friendly companies
- **Tools:** ADHD/ASD apps, products (affiliate)
- **Community:** Discord, meetups, resources

---

## Target Audience

- Late-diagnosed ADHD/AuDHD/autistic BIPOC adults (ages 18-45)
- Providers seeking to serve this community

## Features (MVP)

1. **Provider Directory**
   - Search: location, specialty, insurance, language, identity
   - Filter: "BIPOC-affirming verified"
   - Profile: bio, specialties, credentials

2. **Basic Listings**
   - Tools & apps (affiliate)
   - Products (sensory, ADHD tools)
   - Community resources

3. **Reviews & Verification**
   - User-submitted reviews
   - Provider verification process

---

## Verification Workflow

1. Provider submits listing → custom "BIPOC-affirming" field
2. Admin review → pending status
3. Manual verification → approved/denied
4. Badge displayed → listing goes live
5. User reviews → trust signal

**Add-ons required:**
- Claim Listings ($99)
- Pricing Manager
